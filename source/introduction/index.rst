
Introduction
============

JSonSails is a Web Application Framework (WAF), which is a software
framework that supports the development of web applications. Web frameworks
aim to alleviate the overhead associated with common activities performed
in development and maintenance of web applications.  The following sections
explain how JSonSails is unique as a WAF.
