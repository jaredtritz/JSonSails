Module IDE
==========

.. warning:: Be careful to save often! There is no automatic save in the IDE

The Integrated Development Environment runs in the web browser (Chrome only)
and enables configuration of existing modules as well as creating new modules 
by coding them in raw HTML, JS, CSS.
